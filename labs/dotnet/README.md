# How to run the labs

## Open individual Lab folders in VS Code

Example from `.../dotnet` directory.

## Mac/Linux example

```console
pwd
# .../agent-framework-dev-day/labs/dotnet
cd lab0/lab
code .
```

From there you can edit, debug, etc. When done, go to next.

```console
cd ../../lab1/lab
code .
```

Onward!
